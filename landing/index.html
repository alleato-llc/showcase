A themeable portfolio engine

List your projects in one clean, resizable table — each with an emoji, a headline, and links. Swap themes and fonts from a gear. Edit entries from a CLI or a desktop app. Your content lives in a tiny instance repo; this engine renders it.

See the live demo View source

What it gives you

◧One table, well done

Emoji · title · headline · links. Live/Source pills, multi-link rows, and segmented pills for the same tool in several languages.

⚙Themeable

A gear switches color themes and fonts. Define your own palettes (and web fonts) — they're just data.

⌘Two editors

A Go showcase CLI (add/edit/reorder/validate) and a Wails desktop GUI with drag-reorder, an emoji picker, a theme editor, and live preview.

⧉Engine + instance

The engine is the template & tool; your portfolio is a thin sibling repo of projects.json + config.json + themes.json.

↔Resizable

Drag the sheet's border to resize it (40–85%); double-click to reset. Small flourishes, no framework.

⚡Static & fast

Astro output — plain HTML/CSS, flash-free theming, deploys to any static host (S3 + CloudFront here).

How it works

  1. Fork/scaffold an instance repo — it holds projects.json, config.json, themes.json.
  2. Edit entries with the CLI or the desktop GUI (or by hand — it's validated JSON).
  3. The engine builds your data into a static site; deploy the dist/ anywhere.
